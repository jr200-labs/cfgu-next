# cfgu-next

`cfgu-next` is the maintained `jr200-labs` fork of [Configu](https://github.com/configu/configu): tooling for managing software configuration data such as environment variables, secrets, and feature flags.

It is published to npm as [`@jr200-labs/cfgu-next`](https://www.npmjs.com/package/@jr200-labs/cfgu-next).

```sh
pnpm dlx @jr200-labs/cfgu-next
```

## Fork status

This repository was forked from [configu/configu](https://github.com/configu/configu) on **22-June-2026**. It retains the upstream project history and is licensed under Apache-2.0.

The fork adds:

- SOPS support
- YAML configuration support

The intent is to keep dependencies current and publish maintained releases of `@jr200-labs/cfgu-next` to npm.
